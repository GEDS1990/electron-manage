# vue-electron-manage

> 一个使用vue-electron构建的管理应用
* 模仿win8来创建的管理界面
* 使用vue-router的导航守卫进行跳转权限判断
* 使用vuex进行登录状态的管理
* 使用element-ui来进行页面的构建
* 使用echarts进行图表的制作
* 使用less进行样式设置

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
